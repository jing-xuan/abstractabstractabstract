# Instructions to run

1. `cd abstractabstractabstract`
2. `yarn`
3. `yarn start`
4. Navigate to http://localhost:8888/ on browser

# Sample programs

```
  function f(){
    f();
  }
  f();
```

```
  function f(x){
    f(x+1);
  }
  f(7);
```

Note that `MIN_NUM` = -10 and `MAX_NUM` = 10. This can be changed at the bottom of `abstract.js`
